# Rf-finder 
1. Prendre des photos et les mettre dans un dossier "data" à la racine
2. Lancer le script end_to_end_process qui va, pour chaque photo dans le dossier data, essayer de trouver toutes les plaque de rf présentes et ainsi associer le chemin du fichier à un ou plusieurs rf.
3. Ouvrir app/index.html pour chercher les photo de chaque RF
